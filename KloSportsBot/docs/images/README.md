# docs/images

Only **redacted or synthetic** screenshots belong here — never a real pool screenshot that
shows members' names or their predictions (that's other people's private data; see
[`../../SECURITY.md`](../../SECURITY.md)).

The repo's `.gitignore` ignores `*.png` everywhere by default and re-includes only
`docs/images/*`, so any image you place here must be deliberately added and verified clean
first.

Suggested additions when you have them:
- A redacted kickoff card screenshot (member names blurred/replaced).
- A synthetic leaderboard with fake names.
- An architecture diagram (no personal data).
